# init-vue3-vite-ts

## 技术栈

```text
Vue3+Vite+Typescript+AnDesignVue+Less+Axios+Pinia
```

## 环境准备

```text
node 版本：18.19.1
```

## 依赖安装

```bash
npm install
or
yarn
```

## 启动

```bash
npm run dev
or
yarn dev
```

## 打包

```bash
# 正式环境打包
npm run build:prod
or
yarn build:prod


# 测试环境打包
npm run build:test
or
yarn build:test


# 本地开发环境打包
npm run build:dev
or
yarn build:dev
```
